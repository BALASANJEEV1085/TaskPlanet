## ğŸ› ï¸ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/BALASANJEEV1085/TaskPlanet.git
    cd TaskPlanet
    ```

2.  **Install Frontend Dependencies**
    ```bash
    cd frontend
    npm install
    # Create .env file with VITE_API_URL=http://localhost:5000/api
    npm run dev
    ```

3.  **Install Backend Dependencies**
    ```bash
    cd backend
    npm install
    # Create .env file with MONGODB_URI, JWT_SECRET, CLOUDINARY_XXX
    npm start
    ```
